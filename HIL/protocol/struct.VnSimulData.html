pub fn packet_to_vn(&self) -> Vec<u8> ⓘ

Converts the structure data into a 130-byte VectorNav binary packet.

The first 8 bytes are headers (Sync, Group selection, etc.). See the VN Control Center to verify these header bytes.

§Return

Returns a Vec<u8> of length 130 ready for serial transmission.
